# 가족 스케쥴 앱 - 명칭 통일 및 이모티콘 피커, 업로드 기능 개선 계획

사용자님의 3가지 요청을 완벽하게 반영하기 위한 구현 계획입니다.

## Proposed Changes

### 1. "구성원" 명칭 통일
- 현재 방명록(FamilyBoard.jsx)의 상단 탭에 **"구성원 관리"**로 되어 있는 라벨 텍스트를, 스케쥴 탭과 동일하게 **"구성원"**으로 수정합니다.

### 2. 업로드 버튼 위치 추가 (갤러리 외부 노출)
- 기존에는 `전체보기 팝업` 내부에 들어가야만 [사진 업로드] 버튼이 있었으나, 사용성을 높이기 위해 직관적인 위치로 꺼냅니다.
- **방명록 시트 (FamilyBoard.jsx) 상단**: `해인이 갤러리` 옆 `[전체보기]` 버튼 바로 우측에 **`[+ 사진 업로드]` 버튼**을 배치하겠습니다. 이렇게 하면 갤러리 팝업을 열지 않고도 메인 화면에서 바로 사진과 동영상을 올릴 수 있습니다.

### 3. 이모티콘 피커 (Emoji Picker) 기능 추가
- 현재는 텍스트 입력창으로 이모티콘을 한 글자씩 직접 입력(`<input type="text" />`)하게 되어 있어 모바일 환경 등에서 설정이 번거롭습니다. 이를 터치 기반의 **이모티콘 전용 선택기(Picker)**로 변경합니다.
- **카테고리 구성**: 사용자가 요구하신 대로 **2가지의 특성 있는 탭**으로 나누어 제공합니다.
  - **탭 1 (자연/풍경 등) 🌸**: 🌸, 🌷, 🌻, 🍀, 🌙, ⭐, ⛅, 🍎, 🍓, 🍒 등
  - **탭 2 (얼굴/사람 등) 👨**: 👨, 👩, 👵, 👴, 👦, 👧, 👶, 😊, 😎, 😆, 😍 등
- **UI 흐름 작동 방식**:
  1. 구성원 설정에서 이모티콘 영역을 클릭합니다.
  2. 새로운 **이모티콘 선택 팝업창**이 뜹니다.
  3. 상단 탭에서 카테고리를 이동하며 원하는 이모티콘을 클릭(터치)합니다.
  4. 클릭하는 즉시 팝업창이 닫히며 해당 구성원의 이모티콘이 업데이트됩니다.

> [!TIP]
> **컴포넌트 분리 (Refactoring)**
> 스케쥴러(`Scheduler.jsx`)와 방명록(`FamilyBoard.jsx`) 양쪽 모두에 구성원 설정 화면이 중복해서 작성되어 있습니다. 이번에 이모티콘 피커라는 다소 복잡한 기능이 추가되므로, 설정 화면 코드를 `MemberSettings.jsx` (또는 동일한 컴포넌트)로 분리 통합하여 어느 시트에서 수정하더라도 코드가 일관되게 적용되도록 구조화하는 작업도 병행하겠습니다.

계획이 마음에 드시면, 승인해주시는 즉시 이모지 피커 설계 및 리팩토링 작업을 시작(Execute)하겠습니다!
