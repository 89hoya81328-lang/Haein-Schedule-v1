# 가족 앱 확장 및 사용성 개선 기획안

사용자님께서 요청하신 6가지 피드백을 모두 반영하기 위한 세부 구현 계획입니다.

## 6대 개선 과제 계획

### 1. 스케쥴 특이사항(메모) 데이터 초기화 🧹
- **Scheduler.jsx**: 시작할 때 기본으로 입력되어 있던 `notes` 데이터들(예: '치과 검진일', '소풍' 등)을 모두 비웁니다. 번거롭게 지우지 않고 바로 작성 가능하도록 정리하겠습니다.

### 2. 오늘 날짜 빨간색 동그라미 하이라이트 🎯
- **Scheduler 주간 뷰 & 달력 뷰**: 앱을 켰을 때 실제 '오늘' 날짜를 감지하여, 채점지에서 동그라미 친 듯한 예쁜 손글씨 폰트 느낌의 붉은색 원형 포인트(`border`)를 날짜 숫자에 그립니다. 날짜가 바뀌어도 자동으로 오늘 날짜를 추적합니다.

### 3. 주말 및 공휴일 일정(가족 일정) 수정 허용 ✏️
- **Scheduler.jsx**: 현재 토/일/공휴일에는 '집콕 휴식', '동물원 나들이' 등의 고정 텍스트가 표시되면서 편집이 불가능하게 잠금 처리 되어 있습니다.
- 특이사항 작성 프로세스와 동일하게, 이 텍스트를 클릭 시 **[가족 일정 수정 팝업]**이 떠서 자유롭게 내용을 변경할 수 있도록 활성화하겠습니다.

### 4. 사진/방명록 - 글 수정 기능 📝
- **FamilyBoard.jsx**: 작성된 가족 방명록 메시지 옆에 연필(수정) 아이콘을 추가합니다.
- 클릭 시 인라인 텍스트 박스로 변해 기존 글을 고치고 다시 저장(V)할 수 있도록 구현합니다.

### 5. 미디어(사진/영상) 수정 및 꾸미기 모드 🎨
- **FamilyBoard 갤러리**: 업로드한 사진과 영상은 현재 슬라이드만 가능한데, 하단에 **[수정/꾸미기]**, **[삭제]** 버튼을 추가합니다.
- **꾸미기 기능**: [수정/꾸미기]를 누르면 사진에 필터(화사하게, 흑백, 세피아, 대비 증가 등)를 이쁘게 입히거나, 하단 설명글(Caption) 부분을 수정할 수 있는 전용 에디터 모달을 띄워드립니다.

### 6. 장바구니 '쿠팡(🚀)' 뱃지 마킹 🛒
- **ShoppingList.jsx**: 항목을 새롭게 추가할 때 `[ ] 쿠팡 주문` 체크박스를 도입합니다.
- 기존에 추가된 항목이라도 리스트 항목 옆의 버튼을 눌러 **'🚀로켓' 마크(쿠팡 마크)**를 껐다 켰다 할 수 있게 만들어, 오프라인 마트에서 살 것과 쿠팡에서 살 것을 시각적으로 완벽하게 분리해드립니다.

---

> [!WARNING]  
> **사진 필터 기능 안내**: '꾸미기' 기능은 브라우저에서 제공하는 강력한 CSS 필터를 사진에 덧칠하는 방식으로 구현하여 빠른 메모리에 가볍게 돌아가도록 작업하겠습니다 (인스타 필터 방식).
